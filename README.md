## Instructions

Add a .env file and add this code with your Alpaca credintals to make an API call via Alpaca SDK

```
 * ALPACA_API_KEY = "your-alpaca-api-key"
 * ALPACA_SECRET_KEY='your-alpaca-secret-key'
```
